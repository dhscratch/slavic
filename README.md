## Installation

Add this line to your application's Gemfile:

    gem 'slavic'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install slavic

## Usage

To use slovo module of the gem add the following lines:

    gem 'slavic'
    Slavic.slovo
